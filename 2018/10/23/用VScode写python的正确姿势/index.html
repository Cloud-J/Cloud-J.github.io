用VScode写python的正确姿势

发表于 2018-10-23
字数统计: 493

当你点进这篇文章的时候，想必已经安装好VScode并添加了python插件了，如果还没有，可以先去看看配置环境

如果你已经配置好环境了，那么接下来我们就开始写第一个程序

首先，创建一个文件夹，随便什么地方，随便什么名字（建议使用用全英文命名文件夹，不然会导致世界毁灭），只要你以后找得到。

比如我现在在桌面上创建了一个名叫”my_code”的文件夹，右键文件夹，不出意外你应该能看到菜单栏上有一项“Open With Code”，它的旁边是VScode的图标。点击这个选项，就可以以这个文件夹作为工作目录打开VScode。

我们看到的界面大概是这个样子红色框框起来的是资源管理器，显示的是打开文件夹的目录结构。“MY CODE”文件夹名，把鼠标移到右边四个选项上会有文字提示，自己去探索一下这四个按钮是干什么吧。
