Anaconda

简介

Anaconda指的是一个开源的python发行版本，其包含了conda、Python等180多个科学包及其依赖项。

下载Anaconda3安装包

  • Anaconda3（Windows）
  • Anaconda3（macOS）
  • 清华大学镜像(Windows 3-5.2.0版本)

安装流程

  1. 找到安装包的文件位置，双击打开安装程序，这里next就行
  2. 选择给系统的那些用户使用，一般直接选“Just Me”就行，如果有多个用户且都要用到就选“All Users”（选择这个选项需要管理员权限）
  3. 选择安装路径（图片中的是我的默认安装路径）
  4. 在“Advanced Installation Options”中第一个选项是”添加Anaconda到我的环境变量“。安装程序不建议我们打钩这个选项，但我们偏偏要打钩（皮）。第二个选项是默认安装python3.6。除非你想要使用多个版本的Python，不然就把这个选项√上。然后点击”Install”
  5. 进入“Thanks for installing Anaconda!”界面则意味着安装成功，点击“Finish”完成安装。

Visual Studio Code

简介

Visual Studio Code (简称 VS Code / VSC) 是一款免费开源的现代化轻量级代码编辑器，支持几乎所有主流的开发语言的语法高亮、智能代码补全、自定义热键、括号匹配、代码片段、代码对比 Diff、GIT 等特性，支持插件扩展，并针对网页开发和云端应用开发做了优化。软件跨平台支持 Win、Mac 以及 Linux。

安装流程

安装包地址：Visual Studio Code

启动安装程序后根据提示一直下一步就好了

配置环境

安装完成后，打开VS Code，在扩展中安装python插件。安装完插件后重启VS Code就可以使用了。
