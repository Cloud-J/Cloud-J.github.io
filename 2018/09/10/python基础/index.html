语法基础

变量

变量的概念基本上和代数中方程变量是一致的，只不过在计算机程序中，变量的类型不只是数字，也可以为任意的数据类型。

标识符

标识符是指Python中允许作为变量名或者其他对象名称的有效符号。标识符的命名规则主要有以下三条：

  • 首字符是字母或下划线
  • 其余可以是字母、下划线、数字
  • 大小写敏感（例如python和Python是不同的标识符）

除此之外，还有一些约定俗成的规则，比如说

  • 全大写的变量名一般被认为是一个常量
  • 以下划线开头的标识符如，_xxx, __xxx。因为下划线对解释器有特殊的意义，一般来说这样的变量名被看做是“私有的”，在模块或类外不能被使用
  • 不要用汉语拼音作为标识符

保留字（也称关键字）

保留字是Python语言的关键组成部分，不可以作为其他对象的标识符。在IDE中常常以不同的颜色表示。

Python 3.X中的保留字有：

‘False’, ‘None’, ‘True’, ‘and’, ‘as’, ‘assert’, ‘break’, ‘class’, ‘continue’, ‘def’, ‘del’, ‘elif’, ‘else’, ‘except’, ‘finally’, ‘for’, ‘from’, ‘global’, ‘if’, ‘import’, ‘in’, ‘is’, ‘lambda’, ‘nonlocal’, ‘not’, ‘or’, ‘pass’, ‘raise’, ‘return’, ‘try’, ‘while’, ‘with’, ‘yield’

运算符

运算符是用来连接操作数的符号，比如“+”、“-”等

python支持以下类型的运算符：

  • 算术运算符
  • 比较（关系）运算符
  • 赋值运算符
  • 逻辑运算符
  • 位运算符
  • 成员运算符
  • 身份运算符

赋值

Python语言的赋值不同于一些其他语言，Python是一种动态的强类型语言，所以变量的赋值是不需要显示声明，而是根据“值“来确定类型

  • 增量赋值 如，m += 1等价于m = m + 1; k = 2等价于 k = k\2
  • 链式赋值 如， PI = pi = 3.1415926
  • 多重赋值 如， x,y = 1,2 等价于 x=1 y=2; x,y = y,x 等价于 x,y 的值进行交换

语句

语句就是可以完整执行一个任务的一行逻辑代码

如，print(“hello world”)就完成了一个在屏幕上输出“hello world”的任务

基础数据类型

数字（Number）

Python 支持三种不同的数值类型：

  • 整型(Int) - 通常被称为是整型或整数，是正或负整数，不带小数点。Python3 整型是没有限制大小的，可以当作 Long 类型使用，所以 Python3 没有 Python2 的 Long 类型。
  • 浮点型(float) - 浮点型由整数部分与小数部分组成，浮点型也可以使用科学计数法表示（2.5e2 = 2.5 x 102 = 250）
  • 复数( (complex)) - 复数由实数部分和虚数部分构成，可以用a + bj,或者complex(a,b)表示， 复数的实部a和虚部b都是浮点型。

字符串（str）

字符串的表示可以用单引号、双引号和三引号（三个单引号）

Python对字符串的操作有非常丰富的函数，在此就不一一说明，想了解的可以参考菜鸟教程或者Python 3.5.2官方文档。

布尔型（bool）

布尔是整数的一个子类型,它的返回值为True或者False.

下面是布尔运算:

  1. x or y => 一者真即为真
  2. x and y => 一者假即为假
  3. not x=> 真为假，假为真

备注：

  1. and 与 or 均为短路运算符,若第一个元素已经满足运算符的判断条件，则不会进行第二个运算符的计算。
  2. not 是一个低优先级的布尔运算符，所以 not a == b 解释为 not (a == b)，且 a == not b 是一个语法错误。

基础数据结构

  • 列表
  • 字典
  • 元组
  • 集合

列表（list）

python中的列表是可变类型，用方括号“[]”来界别，并且可以通过索引来访问到类表中的元素

  1. 索引从0开始累计
  2. list中的元素不需要具有相同的类型
  3. 可以使用语法糖[]快速构建一个list

以下是一些常见的对列表的操作：

a = [1, 3, 8, '1', 1] # 用语法糖[]构建列表

a[0] = 2 # 将列表中第0个元素改为2

a.append(5) # 在列表的末尾加上 5 这个元素

a.count(1) # 统计元素1出现的次数

a.insert(2,1) # 将元素1插入到索引为2的位置

更多关于列表的方法可以参考菜鸟教程-列表

元组（tuple）

从功能上看，元组和列表的区别是，列表可以被修改，而元组不可以。比如

a = [4, 5, 6]

a[0] = 1

是可以的，而

a = (4, 5, 6)

a[0] = 1

显然是会报错的。

虽然元组中的元素值是不允许修改的，但我们可以对元组进行连接组合 ，比如可以用+来连接两个元组

关于元组的操作，在菜鸟教程-元组中也有详细的解释

字典(dictionary)

字典是另一种可变容器模型，且可存储任意类型对象 。数学上来讲，他实际是一个映射，通俗来讲，字典也相当于一个列表，只不过它的“下标”是自己定义的“键”，每个“键”对应一个“值”，这样的形式称为“键值对”。键和值之间用:来分割，每个键值对之间用,隔开，整个字典用{}来界别，下面是一个例子：

dict1 = {key1: value1, key2: value2}

在一个字典中键一般是唯一的，值不需要唯一，如果同一个键被赋值多次，则最后一个值会被记住。

集合(set)

集合这一概念和数学上的集合的概念基本一致，与列表区别在于

  • 元素是不重复的，而且是无序的
  • 不支持索引
  • 可以执行数学意义上的交（并）集

我们一般用{}或者set()函数来创建一个集合

关于集合的运算

a = t | s # 并集

b = t & s # 交集

c = t - s # 差集（在t中，但不在s中）

d = t ^ s # 对称差集（在t或s中，但不同时在两者之中）

控制语句

条件语句(if…elif…else)

关于if条件语句可以参考廖雪峰博客或者菜鸟教程。

注意：

  1. 使用if语句时不能忘记:。
  2. if语句是遵循自上而下判断，即如果if条件成立，后面的elif判断语句将被忽略。
  3. if语句与代码块间有4个空格的缩进关系。

循环语句(for/while)

关于循环语句可以参考廖雪峰博客或者菜鸟教程。

注意：

  • 无限循环时是while True: 而不是while 1:
  • 在测试过程中，若出现代码死循环可以使用ctrl+c强制结束程序。

函数

Python用def来定义函数。

与一般语言不同的是，python的函数可以有多个返回值

此外，python支持用lambda表达式来定义“匿名函数”

其中函数的参数分为：位置参数、默认参数、可变参数、关键字参数。详细信息请参考廖雪峰博客。

如果对某个函数的参数不清楚，可用内置函数help()显示其帮助文档。

若非交互式环境，还需调用print()函数。

需要注意的是，成员的访问是有作用范围的

在Python中，通过一套命名体系来识别成员的访问范围。

  • 所有的以字母开头的成员名称被python命名体系自动识别为public
  • 单个下划线开头的成员（如 _foo）被识别为protected，只有类对象和子类对象自己能访问到这些变量，不能用“from xxx import *”而导入
  • 双下划线开头的成员（如__foo）被识别为private，只有类对象自己能访问，连子类对象也不能访问到这个数据
  • 以双下划线开头和结尾的（__foo__）代表python里特殊方法专用的标识，如 __init__（）代表类的构造函数。

编码与解码

decode（解码）

decode的作用是将其他编码的字符串转换成Unicode编码，如str1.decode(‘utf-8’)，表示将utf-8编码的字符串转换成Unicode编码。

encode（编码）

encode的作用是将Unicode编码转换成其他编码的字符串，如str2.encode(‘utf-8’)，表示将Unicode编码的字符串转换成utf-8编码。

注意: 在没有特殊业务需求时，强烈建议只使用UTF-8编码

关于字符串编码的详细资料可以参考廖雪峰博客
